  Textile Engineering Production Analysis using Machine Learning

1. Project Overview
This project applies machine learning to textile production data for:
- Defect prediction
- Predictive maintenance
- Process optimization
- Energy analysis
- Alert systems

2. Dataset
https://www.kaggle.com/datasets/zara2099/textile-engineering-production-dataset

3. Structure
- Notebooks:7 research question notebooks
- results/tables: CSV outputs
- results/figures: PDF outputs


4. Libraries
pandas, numpy, matplotlib, scikit-learn, xgboost
